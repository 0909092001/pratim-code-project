
# Vendor Performance Data Analytics

## Overview
This project demonstrates an end‑to‑end data analytics workflow to evaluate vendor performance using SQL, Python, and Power BI.

## Business Problem
Organizations work with multiple vendors, but inconsistent performance can impact cost, delivery, and profitability. This project analyzes vendor data to identify performance gaps and improvement opportunities.

## Tech Stack
- SQL – Data cleaning and transformations
- Python – Exploratory Data Analysis (EDA)
- Power BI – Dashboards and KPIs
- PDF – Business reporting

## Project Workflow
1. Data ingestion and cleaning using SQL
2. Exploratory analysis using Python
3. Dashboard creation in Power BI
4. Business reporting and recommendations

## Key Metrics
- Total Sales by Vendor
- Profit Margin Percentage
- Inventory Turnover
- Vendor Contribution Analysis

## Outcome
The analysis helps decision‑makers:
- Identify top and weak vendors
- Optimize procurement strategies
- Improve vendor negotiations
